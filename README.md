# diploma_react_coffee_project

Modern starter for students: **React 19 + Vite + React Router + Sass + Vitest**.

## Requirements
- Node.js 22 LTS recommended
- npm 10+

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Tests
```bash
npm run test
npm run test:run
```
